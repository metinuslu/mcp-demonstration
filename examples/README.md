# Weather

## Commands
- Create a new directory for our project  
uv init examples  
cd examples

- Create virtual environment and activate it  
uv venv  
.venv\Scripts\activate  

- Install dependencies  
uv add mcp[cli] PyPDF2

- Create our server file  
touch `FILENAME.py`  

## MCP Inspection  
mcp dev `FILENAME.py`  

## Running the server
uv run `FILENAME.py`
